# trademark-search

Search the USPTO trademark database to check name availability and get registration details. Uses web search, USPTO's TSDR API, and third-party databases for comprehensive trademark research.

## Installation

Copy the `trademark-search` folder into your agent's skills directory.

## Requirements

No API key required for basic searches. An optional USPTO API key enables richer TSDR data access.

## Usage Examples

### Check if a name is trademarked
> "Is 'Stellar' trademarked?"

### Research availability for a new product
> "Can I use 'NovaPay' for a fintech app?"

### Check existing registration status
> "What's the status of trademark registration #5678901?"

## Data Sources

- [USPTO Trademark Search](https://tmsearch.uspto.gov/) — Official US trademark database
- [USPTO TSDR](https://tsdr.uspto.gov/) — Trademark status & document retrieval
- [Trademarkia](https://www.trademarkia.com/) — Third-party trademark search
- [WIPO Global Brand Database](https://branddb.wipo.int/) — International trademarks

## Disclaimer

This skill provides informational data only — not legal advice. Always recommend consulting a trademark attorney for definitive guidance.
