# motion-design-patterns

Framer Motion (Motion) animation patterns for React — springs, staggers, layout animations, micro-interactions, scroll effects, and page transitions. The difference between "it works" and "it feels great."

## Installation

Copy the `motion-design-patterns` folder into your agent's skills directory.

## What's Inside

- **8 core patterns** with copy-paste code: fade+rise, staggered lists, layout animations, exit animations, hover/tap, scroll-triggered, page transitions, number transitions
- **Spring configuration guide** with recommended defaults for every use case
- **Anti-patterns** to avoid — common mistakes that make animations feel cheap
- **Quick decision guide** — which pattern + spring config for any scenario
