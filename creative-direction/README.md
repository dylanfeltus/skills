# creative-direction

Image prompt templates, model selection guidance, and anti-generic patterns for AI-generated visual assets. Stop getting "a person using a laptop" and start getting images with actual creative direction.

## Installation

Copy the `creative-direction` folder into your agent's skills directory.

## What's Inside

- **Prompt templates** for every asset type: hero images, feature illustrations, testimonials, OG cards, icons, backgrounds
- **Model selection guide** — when to use DALL-E 3 vs Gemini vs Midjourney vs Flux vs stock photos
- **Anti-generic playbook** — the specificity ladder, overused tropes to avoid, realism techniques
- **Style consistency framework** — how to keep multiple images cohesive
- **Ready-to-paste prompts** for common scenarios

## Usage Examples

### Get a hero image
> "I need a hero image for a project management SaaS"

### Generate consistent illustrations
> "Generate feature illustrations for our 4 main features"

### Fix generic images
> "Our AI images look too generic, help"
